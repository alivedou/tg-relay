# tg-relay-bot 增强路线图

> 目标：面向 Discord 社区免费服务器面板（Pterodactyl 等）的 Python 服务器模板，拖入即跑。

## 核心设计原则
- Flask 内嵌 HTTP 健康检查（免费容器必须绑定端口）
- 环境变量驱动一切配置，零硬编码
- SQLite 兜底存储，不依赖外部数据库
- Polling / Webhook 双模式自适应

---

## 第一档：部署适配 ✅

| # | 功能 | 说明 | 状态 |
|---|------|------|:--:|
| 1 | Flask 健康检查服务器 | `GET /health` → 200 + JSON，解决免费容器杀进程 | ✅ |
| 2 | Polling / Webhook 双模式 | `TG_WEBHOOK_URL` 存在走 Webhook，否则走 Polling | ✅ |
| 3 | 全配置环境变量化 | 所有配置环境变量读取，移除硬编码 | ✅ |
| 4 | 优雅日志 | 统一 `logging`，时间戳 + 级别 + 脱敏 | ✅ |
| 5 | 启动自检 | 验证 Token 可用性、Owner ID 合法性 | ✅ |
| 6 | 依赖补全 | `requirements.txt` 加入 `flask` | ✅ |
